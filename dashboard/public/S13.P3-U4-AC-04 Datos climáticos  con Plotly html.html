1. Introducción

En el análisis de datos meteorológicos y cíclicos, los gráficos tradicionales a menudo fallan al representar direcciones (ángulos) o periodicidad. Plotly ofrece herramientas potentes para crear Gráficos Polares, que utilizan un sistema de coordenadas circulares donde cada punto se determina por una distancia al centro (radio) y un ángulo.

En esta práctica, exploraremos el dataset "Wind" para representar la fuerza y dirección del viento mediante tres variantes interactivas: dispersión, líneas y barras (rosas de los vientos).

4. Desarrollo de la Práctica Paso a Paso

Paso 1: Carga de Datos Climáticos

Plotly incluye datasets de prueba. Utilizaremos wind(), que contiene información sobre la dirección del viento y su intensidad.

import plotly.express as px

# Cargar el dataset integrado
wind_df = px.data.wind()

# Inspección inicial
print(wind_df.head())
print(wind_df.info())

Nota: Observa las columnas direction (dirección cardinal), strength (fuerza) y frequency (frecuencia de ocurrencia)[cite: 16, 18].

Paso 2: Gráfico de Dispersión Polar (Scatter Polar)

Este gráfico es ideal para ver la distribución de puntos individuales. Usaremos r para la magnitud y theta para la dirección angular.

fig1 = px.scatter_polar(wind_df, 
                        r="strength", 
                        theta="direction", 
                        color="strength",
                        symbol="direction",
                        size="frequency",
                        title="Dispersión Polar: Fuerza vs Dirección del Viento")
fig1.show()

Paso 3: Gráfico de Líneas Polar (Line Polar)

Útil para observar tendencias continuas o cierres de ciclos. En este caso, conectaremos los niveles de fuerza por dirección.

fig2 = px.line_polar(wind_df, 
                      r="strength", 
                      theta="direction", 
                      color="strength",
                      line_close=True, # Cierra el círculo de la línea
                      title="Tendencia Polar: Intensidad por Dirección")
fig2.show()

Paso 4: Gráfico de Barras Polar (Wind Rose)

Este es el estándar para representar la dirección del viento (Rosa de los vientos). Las barras se extienden desde el centro según su magnitud.

fig3 = px.bar_polar(wind_df, 
                    r="frequency", 
                    theta="direction", 
                    color="strength",
                    color_discrete_sequence=px.colors.sequential.Plasma_r,
                    title="Rosa de los Vientos: Frecuencia por Dirección")
fig3.show()

Paso 5: Personalización Avanzada y Exportación

Ajustaremos la apariencia del gráfico para un reporte profesional y guardaremos los resultados como archivos HTML interactivos.

# Ejemplo de personalización del diseño
fig3.update_layout(
    font_size=12,
    legend_font_size=10,
    polar_angularaxis_rotation=90, # Rota el gráfico 90 grados
)

# Guardar como HTML
fig1.write_html("polar_scatter.html")
fig2.write_html("polar_line.html")
fig3.write_html("polar_bar.html")
